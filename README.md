# Smart Study Planner

A Java-based Smart Study Planner developed as a university Object-Oriented Programming (OOP) project. The application features a graphical user interface (GUI) built using Java Swing to help users manage their study tasks efficiently.

## Features

- Add exam tasks
- Add assignment tasks
- View all tasks
- Delete tasks
- User-friendly Java Swing GUI
- Input validation with dialog messages

## OOP Concepts Used

- Classes and Objects
- Encapsulation
- Inheritance
- Polymorphism
- Constructors

## Technologies Used

- Java
- Java Swing
- ArrayList
- VS Code

## About the Project

This project was developed as part of a university OOP course to demonstrate the practical implementation of Object-Oriented Programming concepts in Java. The application provides a simple graphical interface that allows users to add, view, and delete exam and assignment tasks.
